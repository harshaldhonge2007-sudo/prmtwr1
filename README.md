# ElecGuide - Election Process Education Assistant

An interactive web application to educate users on the election process in India.

## Features
- **AI Chat Assistant**: Powered by Google Gemini.
- **Election Timeline**: Interactive view of election phases.
- **Voting Guide**: Step-by-step instructions for voters.
- **FAQ**: Frequently asked questions.
- **Responsive Design**: Mobile-first, glassmorphism UI.

## Tech Stack
- **Frontend**: React / Vanilla JS (Production build)
- **Backend**: Node.js / Express
- **Deployment**: Google Cloud Run
